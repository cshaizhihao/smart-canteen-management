# 校园食堂智慧化管理系统 (Smart Canteen Management System)

🚀 **一个基于现代 Web 技术的高效校园饮食解决方案。**

本系统旨在解决传统校园食堂排队久、信息不透明、财务对账难等问题。通过数字化的订餐流程和实时的库存管理，提升学生的就餐体验与食堂的运营效率。

---

## ✨ 核心特性

- **💻 现代化 UI**: 基于响应式设计，适配手机与电脑端。
- **🔐 多角色支持**:
  - **学生端**: 查看今日菜单、在线下单、余额即时扣费、追踪取餐状态。
  - **管理员端**: 菜品价格/描述修改、实时库存预警、订单流程管理。
- **📊 实时状态追踪**: 从“已下单”到“待取餐”的全程闭环记录。
- **💾 数据持久化**: 使用内嵌 SQLite 数据库，无需安装额外数据库软件，即开即用。
- **💰 充值模拟**: 预设学生账户余额，体验完整的支付流程。

## 🛠️ 快速安装 (Port: 10086)

只需在安装了 **Node.js** 的服务器上执行以下指令：

```bash
# 克隆仓库
git clone https://github.com/cshaizhihao/smart-canteen-management.git

# 进入目录
cd smart-canteen-management

# 安装依赖项
npm install

# 启动应用 (默认监听 10086 端口)
PORT=10086 node index.js
```
